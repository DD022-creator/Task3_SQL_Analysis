# Task 3: SQL for Data Analysis

## Project Overview
This project uses SQL to analyze an e-commerce database with customers, products, orders, and payments.

## Tools Used
- **SQLiteOnline.com** (Online SQLite compiler)

## SQL Concepts Demonstrated
1. SELECT, WHERE, ORDER BY
2. GROUP BY with Aggregate Functions (SUM, AVG, COUNT)
3. INNER JOIN
4. LEFT JOIN
5. Subquery
6. Create View
7. Use View
8. WHERE vs HAVING
